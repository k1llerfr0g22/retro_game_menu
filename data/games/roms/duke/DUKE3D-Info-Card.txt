* Note for DOSBox users *

This software is fully installed and configured to run in the DOSBox emulator.
You can simply drag and drop the file DUKE3D.BAT over the DOSBox icon to start
playing. Experienced DOSBox users may choose other methods of startup.

!!! IMPORTANT !!!

Please DO NOT use dn3d-brw.bat to run the game! This is a special file for the
browser version of DOSBox and will not play Duke Nukem 3D in the best possible
quality.

The software was configured and tested to work with the official DOSBox build
v0.74-3 available from www.dosbox.com. It may or may not work with any other
third-party DOSBox distributions that are available.

For playing instructions and other information, please refer to the
DN3DHELP.TXT file found in the DUKE3D directory.

* Note on the use of S3 VBE/Core 2.0 and S3 Speed Up *

S3 VBE/Core 2.0 and S3 Speed Up are utilities by Dietmar Meschede that improve

S3 VBE/Core 2.0 and S3 Speed Up are free software. Please see the respective
documentation in S3VBE20 and S3SPDUP directories for more information and

* Source ports *

This game can also be played natively (no DOSBox required) on Windows, MacOS,
Linux and other modern platforms using one of the source ports.

> EDuke32 (the most advanced port):
  http://www.eduke32.com/

NOTE: You can download high quality music packs for Duke Nukem 3D to be played
in EDuke32. These tracks were recorded on authentic 90s hardware for which the
Duke Nukem 3D soundtrack was originally written:

http://sc55.duke4.net/games.php#duke3d

You can also download the high-quality sound effects from the PlayStation
version of Duke Nukem 3D to be used with the PC version:

http://hrp.duke4.net/download.php

> Raze (based on EDuke32 code):
  https://github.com/coelckers/Raze/releases

> JFDuke3D (one of the first ports):
  http://jonof.id.au/jfduke3d/index.html

> xDuke (more classic look):
  http://vision.gel.ulaval.ca/~klein/duke3d/
